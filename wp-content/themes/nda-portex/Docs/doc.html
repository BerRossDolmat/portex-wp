1.Техническая часть

Структура Темы

Все фалый темы расположены в wp-content/themes/nda-portex

Общие настройки темы, includes, actions, hooks и shortcodes подключены в файле functions.php в корне темы.

404.php - шаблон страницы ошибки category.php - шаблон страниц категорий и подкатегорий footer.php - универсальный футер сайта. Так же в нем содержится кнопка связаться с нами, модали контактов и связаться с нами, а так же место подключения скриптов. functions.php - файл настройки темы. Содержит в себе настройки тем предпросмотров изображений, и подключение скриптов и стилей ( см. файл enqueue.php из папки includes) header.php - файл хедера. Содержит в себе навигационное меню и его мобильные версии, а так же html head и точку подключения стилей. index.php - основной шаблон темы, отвечает за главную страницу
search.php - шаблон страницы поисковой выдачи. searchform.php - шаблон строки поиска. single-product.php - шаблон продукта. style.css - шайл настройки темы.

Папка assets содержит в себе все файлы, использующиеся в теме, которые не являются php шаблонами темы. css - стили. fonts - используемые шрифты. img - статичные картинки. js - скрипты.
